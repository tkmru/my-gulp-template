# my gulp template
my Gulp task template for web develop with sass

## Requirements

- Node.js
- Gulp.js

## Quickstart

```
$ git clone https://github.com/tkmru/my-gulp-template.git my-project
$ cd my-project
$ npm install
```

## How to Use

start LiveReload

```
npm run livereload
```

compile Sass

```
npm run sass
```

auto Sass compile & LiveReload

```
npm run watch
```
